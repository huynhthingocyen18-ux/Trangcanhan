# Trangcanhan
Thông tin cá nhân
Họ và tên: Huỳnh Thị Ngọc Yến

Chuyên ngành: Đồ Họa Kỹ Thuật Số

Lĩnh vực trọng tâm: Trực quan hóa dữ liệu (Data Visualization), R&D, UI/UX cho hệ thống tự động hóa.

🏆 Thành tựu nổi bật
Trang web được thiết kế để làm nổi bật sự uy tín trong học thuật của tác giả:

16 Bài báo quốc tế hạng Q1: Minh chứng cho khả năng nghiên cứu chuyên sâu.

02 Bằng sáng chế: Thể hiện tư duy sáng tạo và giải quyết vấn đề thực tiễn.

💻 Các dự án chính
Hạ tầng 5G: Tối ưu hóa giao diện quản lý băng thông rộng.

Mạng lưới 6G: Concept trực quan hóa kết nối viễn thông tương lai.

AI Automation: Hệ thống tự động hóa quy trình thiết kế bằng trí tuệ nhân tạo.

🛠 Công nghệ sử dụng
HTML5 & CSS3: Sử dụng CSS Grid (Bento Style) và Flexbox.

JavaScript (Vanilla): Xử lý logic chuyển đổi giao diện (Dark/Light Mode).

Typography: Plus Jakarta Sans (font chữ hiện đại cho thiết kế kỹ thuật số).

Hiệu ứng: Glassmorphism, CSS Animations, và Responsive Design.

🎨 Tính năng UI/UX
[x] Dark/Light Mode: Nút chuyển đổi giao diện mượt mà ở đầu trang.

[x] Bento Grid: Sắp xếp thông tin khoa học, ưu tiên các con số thống kê quan trọng.

[x] Mobile Responsive: Tự động điều chỉnh layout trên thiết bị di động.

[x] Micro-interactions: Hiệu ứng hover và fade-in khi tải trang.
